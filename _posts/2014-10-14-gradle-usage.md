# Install

## Requires:

* jdk/jre 1.6+

## Download

[Gradle web site](http://www.gradle.org/downloads)

## Unpacking

## 设置环境变量

* 设置 GRADLE_HOME

* 将 GRADLE_HOME/bin 加入 PATH 变量

* 设置 GRADLE_USER_HOME (自定义`.gradle`的路径)

## Running and testing

	gradle -v
