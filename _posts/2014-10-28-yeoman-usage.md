# Info

[Yeoman](http://yeoman.io/) 是一种构建 javascript 项目的脚手架。

是 Google 的团队和外部贡献者团队合作开发的，他的目标是通过Grunt（一个用于开发任务自动化的命令行工具）和Bower（一个HTML、CSS、Javascript和图片等前端资源的包管理器）的包装为开发者创建一个易用的工作流。

Yeoman 主要有三部分组成：

* yo（脚手架工具）;
* grunt（构建工具）;
* bower（包管理器）。

这三个工具是分别独立开发的，但是需要配合使用，来实现我们高效的工作流模式。
