# Info

ruby 版本管理工具。

# 安装

	// 安装稳定版本
	curl -sSL https://get.rvm.io | bash -s stable

	// 安装开发版
	curl -sSL https://get.rvm.io | bash

# 使用

	ruby -v
	rvm info
	rvm list
	// 查询可安装的版本
	rvm list known
	// 安装某一版本的ruby（可安装jruby等）。
	rvm install 2.0.0
	rvm [use] 2.0.0
	rvm use default
	rvm use system
	rvm reset
	rvm alias create alias_name 2.0.0
	rvm use alias_name
	rvm alias delete alias_name
	rvm alias list
	rvm remove 2.0.0
	rvm uninstall 2.0.0
