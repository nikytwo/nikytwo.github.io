## Vundle

### Info

[Vundle](https://github.com/gmarik/Vundle.vim) 是 Vim 中的插件管理工具。

### 安装

* Linux

	git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

* Windows


### 设置

### 安装其他插件

* 启动 Vim 并运行 `:PluginInstall`

* 命令行下安装：`vim +PluginInstall +qall`
